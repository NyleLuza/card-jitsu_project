# DojoBlox
Repository generated by Rojo 7.5.1. and GitSync plugins for Roblox Studio. DojoBlox is a Roblox strategy card game where players can collect unique elemental cards and use them in one-on-one battles with friends or rivals around the world! Elements consist of Fire, Snow, and Water. Fire beats snow, snow beats water, and water beats fire. If two cards of the same element go against each other the card with the higher value wins. If elements and values are the same, its a tie.

## Why this Project?
All four of us were interested in game development and we came to the consensus of recreating the iconic game "Card-Jitsu." We are looking to create our own twist on the classic Club Penguin strategy card game with the use of the Roblox Studio environment. Our goal is to create a fully functional game to publish to the Roblox game catalog. 

## Github Version Control not really used
Initially we created this repo with a Roblox Studio plugin called Rojo, but it requires the use of a 3rd party IDE to sync, and we wanted to write scripts natively in Roblox Studio. Because of this, we used ROblox Studio's built in tools for version control and editing, which made the github repo unneccesary. The scripts on this repo are purely for the purpose of assignment submission for our senior design project.
