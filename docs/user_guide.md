# DojoBlox User Guide

## Introduction
This guide is intended for players to understand the objectives of DojoBlox and enjoy the experience as intended by the developers.

DojoBlox is a recreation of the game Card-Jitsu from Club Penguin featured in Roblox. Players battle each other using elemental cards in a turn-based match. The objective of the game is to defeat your opponent and win the indicated number of rounds, encouraging strategic card selection. Outside of matches, players will be able to configure their decks and unlock cards through a gacha system and/or card pack purchases.

## Installation and Setup
1. Install Roblox.
2. Search for DojoBlox in the Roblox game catalog and select it.
3. Click the blue play button to run the experience.

## How to Start a Game
### Ranked Matches
1. Join the experience in Roblox.
2. Wait in the matchmaking queue until an opponent has been found.
3. Both players will be transported to the Battle Arena to begin the match.

### Matches Against Connections (Roblox friends)
1. Join the experience in Roblox.
2. You and your connection must be in the same server for this experience.
3. Use the connections menu to send a match request to your connection.
4. Wait for your connection to accept the match request.
5. Upon being accepted, both players will be transported to the Battle Arena to begin the match.

## How to Play a DojoBlox Match
Each player starts with a 5 card hand.
Each player selects one card per turn.
Cards have an element and a number.
- Fire defeats Snow.
- Snow defeats Water.
- Water defeats Fire.
In the event of both cards having the same element, the card with the higher numeric value wins.
If elements and values are identical, the round is a tie.
Cards may have special rules indicated by the text on the card. These rules can introduce new gameplay mechanics and/or alter the above game rules.

## Controls/Interface
Click or tap a card to select it for play.
Use the confirm button to indicate you are ready for the card comparison.
The game will wait until both players have confirmed before resolving the round.

## How to Create a DojoBlox Deck
Decks must consist of exactly 24 cards.

1. Using the Deck Editor, select a deck from your inventory you wish to edit, or create a new one from scratch.
2. Add or remove cards from your deck as you see fit.
3. After you are finished with you changes, press the save button to update your deck.
4. Select your new deck in your inventory to use it for the next match.

## Game Results
Players win by winning the indicated number of rounds first.
When a match ends, the results screen will display the winner.
In the previous screen, and/or after returning to the lobby, the player will also be presented with any rewards earned from the battle, including ranking changes and items received.

## Troubleshooting
A bug submission feature will be available if any gameplay elements or features appear to be missing or malfunctioning, allowing bug reports to be sent to the developers.

In the event of a continually faulty experience, it is recommended to leave the experience and join the game again in a server different from the faulty one.
